# Pathogen & Microbiome Institute Website Redesign

This project is a complete redesign of the Pathogen & Microbiome Institute (PMI) website at Northern Arizona University, inspired by the modern design and branding of the NAU Nano website.

## Project Overview

The redesigned website will feature:
- Modern, responsive design
- Clean typography and layout
- NAU-inspired color palette (blue and gold)
- Improved user experience and accessibility
- Modular component architecture

## Structure

- `index.html` - Main webpage
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript for interactivity
- `assets/` - Images and other assets

## Getting Started

Open `index.html` in a web browser to view the redesigned website.

## Development

This is a static website redesign project. No build tools are required.
